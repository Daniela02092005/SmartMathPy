# Utils

**Utils** es una carpeta dedicada a los algoritmos básicos de las matemáticas como lo son la suma, resta, multiplicación, exponenciación y
conversión de valores. Sin embargo, esta carpeta tiene un objetivo más didáctico respecto al resto del repositorio, es decir, estos algoritmos se hacen con la idea de sentar una pequeña base en **SmartMathPy** mas no serán usados en los demás algoritmos del mismo.

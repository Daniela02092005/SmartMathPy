# Fundamentos

**Fundamentos** es la carpeta, que como su nombre indica, contiene los algoritmos fundamentales de la matemática: conjutnos, relaciones y funciones. Sin embargo, estos algoritmos no se profundizaron demasiado porque aunque son importantes, no son el centro de el álgebra, el cálculo o la estadística, las cuales son el enfoque de este proyecto.

## Objetivo del módulo

1. Crear algoritmos matemáticos que evidencien la capacidad de convertir definiciones, axiomas y teoremas de conjuntos, relaciones y        funciones en algoritmos.
